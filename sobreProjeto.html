Sobre o Projeto Air&Game

Documentação técnica completa do desenvolvimento do e-commerce de tênis Air&Game, incluindo objetivos, tecnologias utilizadas e metodologias aplicadas.

Objetivos do Projeto

Objetivo Principal

Desenvolver um e-commerce moderno e responsivo para venda de tênis, oferecendo uma experiência de compra intuitiva e agradável aos usuários.

Objetivos Específicos

  • Criar uma interface moderna e atrativa
  • Implementar navegação intuitiva por categorias
  • Criar páginas de produtos com informações detalhadas
  • Garantir responsividade em todos os dispositivos
  • Implementar sistema de contato eficiente

Dificuldades encontradas

  • Ajustes finos de responsividade entre colunas (flexbox vs porcentagens) — por exemplo, controlar padding e box-sizing para evitar overflow em telas menores.
  • Tratamento de imagens (dimensões e object-fit) para que não “quebrem” o layout dos cards.
  • Adapatacao para responsividade aṕos visualização para desktop está pronta.

Pontos Fortes

  • Design Limpo e Objetivo: Interface minimalista focada no produto.
  • Responsividade Robusta: Adaptação eficaz a diferentes tamanhos de tela.
  • Navegação Intuitiva: Estrutura de links clara e fácil de usar.
  • Conformidade com Requisitos: Atendimento a todos os requisitos acadêmicos.

Pontos Fracos

  • Funcionalidade Estática: Ausência de backend (pagamentos, banco de dados real).
  • Conteúdo Fixo: Informações de produtos e notícias estáticas.
  • Personalização Profunda do Bootstrap: Pode ser trabalhosa sem pré-processadores CSS.

Periodicidade de Atualização

Como um projeto acadêmico, a periodicidade de atualização é pontual, focada na entrega dos requisitos. Em um cenário de e-commerce real, um site como este exigiria atualizações contínuas para adicionar novos produtos, notícias, promoções, implementar novas funcionalidades, e realizar manutenção de segurança e performance para garantir a melhor experiência ao usuário e a competitividade no mercado.
